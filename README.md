# Just show my ip (just-show-my-ip)

A simple node.js package that shows host ip address.
The package returns a string with ip address. 

Install:
```shell
npm i just-show-my-ip
```

Usage:
```javascript
const getMyIp = require('getMyIp');

console.log(getMyIp());
```

Result:
```
$ node index.js
143.124.53.145
```
